Allah berfirman sehubungan dengan kisah Nabi Ibrahim a.s.: (Maka tiadalah jawaban kaumnya selain mengatakan, "Bunuhlah atau bakarlah dia," lalu Allah menyelamatkannya dari api) mereka melemparkannya ke dalam api, sedangkan Allah menjadikan api itu dingin dan keselamatan bagi Ibrahim.

(Sesungguhnya pada yang demikian itu) yakni diselamatkannya Nabi Ibrahim dari api (benar-benar terdapat tanda-tanda kebesaran Allah) yaitu tidak berpengaruhnya api terhadap diri Nabi Ibrahim, padahal api itu sangat besar nyalanya, kemudian dalam waktu yang sangat singkat Allah menjadikan bekas api itu sebuah taman (bagi orang-orang yang beriman) yakni bagi orang-orang yang percaya kepada keesaan Allah dan kekuasaan-Nya, karena hanya mereka saja yang dapat mengambil manfaat dari kisah ini.
